# 📖 About

This repository contains all the projects, experiments, assignments, and learning materials developed during the **Summer Boot Camp on Internet of Things (IoT)**.

The goal of this repository is to provide practical knowledge of embedded systems, Arduino programming, sensor interfacing, automation, and real-world IoT applications.

---

# 🎯 Objectives

- Learn IoT fundamentals
- Understand Arduino programming
- Interface sensors and actuators
- Build real-world IoT projects
- Improve hardware and software integration skills
- Develop problem-solving abilities through hands-on practice
